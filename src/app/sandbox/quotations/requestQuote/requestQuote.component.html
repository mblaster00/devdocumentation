This operation is used to get API KEY to access to Logidoo Ecommerce API.

/requestQuote - POST


Try it

Request URL

Request headers

  • string

    Basic authentication header containing API user ID and API key. Should be sent in as HS245 encoded.

  • (optional)
    string
    Media type of the body sent to the API.

Request body

  • application/x-www-form-urlencoded
access_token

Responses

Representations

  • application/json 

{{ '{' }}
    "username": "string",
    "password": "string"
{{ '}' }}

    "password": "pass123"
